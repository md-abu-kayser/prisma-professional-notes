# Prisma vs Other ORMs

- **Prisma**: type‑safe, auto‑generated client, declarative schema.
- **TypeORM**: decorator‑based, both active record and data mapper.
- **Sequelize**: traditional, large feature set.
- **Drizzle**: lightweight, SQL‑like.
